1. 概要
　WindowsCE用スケジューラの定番であるOyajin Appointmentのメニュー及びダイア
  ログの英語のキャプション(テキスト)を和訳したOyajin Appointment日本語版？
　を作ってみました。
　英語より日本語の方が得意な方は是非お試し下さい。
　なお、Oyajin Appointmentのドキュメント(readme.txt)は、英語のキャプション
　を前提とした説明になっているので、初めてOyajin Appointmentをお使いになる方
　は、オリジナルの実行ファイルで使い方をマスタした方が良いかもしれません。


2.注意事項

　・Oyajin Appointmentは中本伸一さんが開発したフリーソフトですが、中本さんは、
　　Oyajin Appointmentの公開を中止し、ソースを含めた全てのリソースをPDSとして
　　公開されています。

　・Oyajin Appointment日本語版？は中本さんが公開したソースを改変して作成しま
　　した。勿論フリーソフトウエアです。

　・プログラム作成の都合で、各CPU別の実行プログラム名は全て app.exeとなって
　　います。

　・私のホームページに「Oyajin Appointment日本語版？」のeMbedded Visual C++
　　3.0用のソースファイルがあります。(注)
　　また、オリジナルのOyajin Appointmentの実行ファイルやソースは「Pocketに
　　書斎」http://homepage2.nifty.com/pocketcafe/ などのサイトから入手できます。
